# BoolBnB

Team project based on the replica of the famous AirBnB site with all its features.

Frameworks used:

- Vue.js 2
- Laravel 7

Libraries:

- braintree (payments)
- axios (HTTP requests)
- tomtom (maps e geo)
- chart.js (statistics graphs)
