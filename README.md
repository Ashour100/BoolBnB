# BoolBnB

Team project based on the replica of the famous AirBnB site with all its features.

See the project in real time: http://bool--bnb.herokuapp.com

Frameworks used:

- Vue.js 2
- Laravel 7

Libraries:

- braintree (payments)
- axios (HTTP requests)
- tomtom (maps e geo)
- chart.js (statistics graphs)
